## 对象扩展

对象扩展机制  extend  mixin

```js
// prototype.js
function extend(destination, source) {
  for (var property in source) {
    destination[property] = source[property];
  }
  return destination;
}
```

问题： 原型方法不会被遍历出来， for in循环无法遍历名为valueOf、toString的属性名
